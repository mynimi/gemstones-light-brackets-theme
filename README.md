# Gemstones Theme Light

After having some trouble with Gemstones in the sun, I decided to turn it into a light theme.
This is the result.

### Installation
* Open Brackets
* Open the Extension Manager
* Switch to "Themes" tab
* Search for "Gemstones Theme Light"
* Click "Install"
